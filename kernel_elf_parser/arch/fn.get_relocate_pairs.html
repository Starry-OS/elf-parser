Read the relocate pairs from the elf file.

§Arguments

  • elf - The elf file
  • elf_base_addr - The base address of the elf file if the file will be loaded to the memory

§Return

It will return a vector of RelocatePair (from super::RelocatePair) which contains the source address and destination address of the relocation.
